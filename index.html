Sobre mim

Sou formado em Sistemas de Informação pela Faculdade de Apucarana (FAP) e tenho 10 anos de experiência no desenvolvimento de software, com foco em Delphi (versões 11, 10.2 e 7) e banco de dados Oracle. Tenho expertise em criação de APIs REST utilizando o framework Horse, além de desenvolver aplicações desktop robustas aplicando conceitos de orientação a objetos, como herança.

Ao longo da minha carreira, implementei soluções que automatizam processos, como a integração entre ERPs e sistemas de marketplace (Bling, Mercado Livre, Magazine Luiza), otimizando vendas e gestão de produtos. Também desenvolvi ferramentas que reduziram custos operacionais ao automatizar tarefas antes manuais.

Atualmente, trabalho como freelancer, aprimorando minhas habilidades em Delphi e expandindo meus conhecimentos em Ciência de Dados com Python. Busco novas oportunidades para aplicar minha experiência em projetos que façam a diferença nos resultados das empresas.

Projeto Desktop Delphi

Este projeto é uma aplicação desktop desenvolvida em Delphi, criada para consumir duas APIs REST desenvolvidas em Delphi: uma para autenticação de usuários e outra para realizar operações CRUD no banco de dados Postgres. A integração entre o cliente desktop e as APIs é feita utilizando bibliotecas modernas para a comunicação HTTP e autenticação JWT. O projeto foi desenvolvido utilizando conceitos de herança, o que permitiu a criação de uma arquitetura mais limpa e modular, promovendo o reuso de código e facilitando a manutenção.

As ferramentas utilizadas foram:

  • RESTRequest4Delphi
  • Horse-JWT
  • BCrypt
